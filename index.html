在台灣，醫療系統每年承受超過 3.6 億次門診，每位病人平均只有 3 分鐘面對醫師。 醫師精確地診斷、開出藥物處方，但離開診間後的生活 —— 飲食、運動、睡眠、社交、心理狀態 —— 幾乎完全脫離醫療體系的視線。

這不是醫療系統的失職，而是結構性的限制。 現有醫療基礎建設只能處理「疾病」，卻無法支撐「健康」。

「我們有全世界最便利的醫療，卻沒有支撐全人健康的數位基礎建設。」
// VISION

數位分身 × 行為處方箋

Project SYNC 打造一個 AI 驅動的健康守護架構，讓醫療從「治療疾病」延伸到「守護健康」， 透過行為設計將健康行為融入日常生活的每個角落。

🧬

全人數位分身

結合穿戴裝置、電子病歷、行為日誌，建構每位使用者的多維健康數位分身。 不只是數據收集，而是將生理、心理、社交、環境整合成可持續追蹤的動態模型。

💊

AI 行為處方箋

AI 分析數位分身資料後，產出個人化的「行為處方箋」——不只告訴你要做什麼， 更設計符合個人生活節奏的行動步驟，讓健康行為像吃藥一樣自然。

🤝

社會處方箋

英國 NHS 驗證的社會處方箋模式 —— 透過社區據點、在地夥伴、同儕支持網絡， 讓健康行為有社交環境的承托，而非孤軍奮鬥。

我們想像的未來是：一位三高的中年上班族，每天早上打開手機收到一份「今日處方」—— 中午走去附近的合作餐廳享受低GI午餐（hMile 折扣）、 下班到社區據點參加30分鐘的伸展課（hEXP 點數累積）。 他的醫師在下次回診前，已經看到持續兩週的改善曲線。

// BEHAVIORAL PRESCRIPTION

讓健康行為像藥物一樣精準

行為處方箋不是衛教單張，而是一套 AI 驅動的個人化行為設計系統。 它分析使用者的生理數據、生活節奏、偏好與能力，產出可執行的行為建議。

  1. 數據收集 穿戴裝置連續記錄心率、睡眠、步數、壓力指標
  2. AI 分析與風險評估 數位分身模型識別偏離健康軌道的信號
  3. 行為處方生成 產出符合 W3C Verifiable Credential 規格的處方箋
  4. 社區驗證與執行 社區據點或合作商家掃碼驗證，完成行為挑戰
  5. 回饋循環 行為完成後獲得 hEXP，AI 模型更新下一週期的處方

社會處方箋源自英國 NHS，核心洞察是：許多健康問題的根源不在醫療，而在社會。 孤獨、缺乏運動夥伴、不知道社區資源 —— 這些比藥物更難開處方。

Project SYNC 將社會處方箋數位化，導入台灣豐富的社區基礎建設。 台灣有超過 4,200 個社區關懷據點、9,600 個里辦公處， 這些場域已經有人力與空間，缺的是數位串連與激勵機制。

社會處方箋的關鍵不是開出處方，而是讓社區的力量被系統看見。

透過 TW DIW（台灣數位身分錢包）與 W3C Verifiable Credentials， 每一次社區參與都成為可驗證的健康憑證 —— 保護隱私的同時，讓健康行為有據可查。

// DUAL COIN ARCHITECTURE

雙幣分離設計

刻意區隔「投入度量測」與「價值流通」兩個層次，避免單一代幣同時承載聲譽與交易功能所導致的激勵扭曲。

SOULBOUND

hEXP

健康經驗值 — 靈魂綁定層
  • 不可轉讓、不可交易
  • 代表「你做了多少」
  • 綁定於數位分身
  • 行為事後鑄造（mint-on-completion）
  • 具衰減機制（停止參與則漸減）
  • 最低保底：歷史峰值 20%
→
達標轉換
單向閘門
CIRCULATING

hMile

健康里程幣 — 流通價值層
  • 可授權、可兌換、可追溯
  • 代表「你的行為值多少」
  • 經行為驗證的注意力憑證
  • 365 天未使用自動銷毀
  • 不可直接變現（非投機工具）
  • 每月轉換上限 50 hMile

為什麼要分兩層？ 如果只有一種幣，使用者會傾向「最小成本獲取最大代幣」而非「最大化健康行為」。hEXP 是一面不會騙人的鏡子——它只反映你真實投入了多少。hMile 必須以 hEXP 為基礎，沒有捷徑。

hMile 健康里程幣

hMile 不是貨幣，不是證券，而是一種經過行為驗證的注意力憑證——持有 hMile 的人，已經用真實的健康行為證明了自己的參與意願，商家為這份注意力買單。

三個供給水龍頭

🔄 hEXP 轉換

使用者端鑄造。100 hEXP = 1 hMile（基礎率），可依健康成效獲得最高 ×2.0 加成。每人每月上限 50 hMile。

活水夥伴以實質承諾金為擔保，1:1 鑄造 hMile 進入獎勵池。基準匯率 1 hMile ≈ NT$2-5。

// BEHAVIORAL SPREAD MODEL

行為價差模型

活水夥伴的核心獲利來自「行為價差」——投入成本與獲得的精準行銷價值之間的差額。

行為價差 = 精準觸及價值 - 獎勵投入成本 - 平台費用 其中： 精準觸及價值 = 觸及人數 × 行為驗證溢價 × 轉換率 行為驗證溢價 = 1.5x - 3x（相對於一般數位廣告）

📐 案例計算：某運動品牌「公園晨跑挑戰」

獎勵投入 1,000 hMile ≈ NT$3,000，平台費用 NT$360（12%），200 人完成任務。

每人觸及成本僅 NT$16.8，但這 200 人都是「有穿戴裝置、持續運動 30 天以上、行為處方箋已完成」的高度活躍者。等效精準投放 CPM：NT$50-80。
